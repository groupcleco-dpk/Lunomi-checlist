⚡ Setup Lunomi Checklist

Hubungkan ke Supabase agar progress tersinkron antara HP dan laptop secara otomatis.

Cara dapat URL & Key:
1. Buka supabase.com/dashboard
2. Klik project Lunomi kamu
3. Klik Settings (ikon gear) → API
4. Copy Project URL dan anon public key

Sebelum connect, buat tabel dulu di SQL Editor:
Lihat SQL yang perlu dijalankan →
CREATE TABLE IF NOT EXISTS lunomi_checklist ( id TEXT PRIMARY KEY, done BOOLEAN DEFAULT FALSE, updated_at TIMESTAMPTZ DEFAULT NOW() ); ALTER TABLE lunomi_checklist ENABLE ROW LEVEL SECURITY; CREATE POLICY "open" ON lunomi_checklist FOR ALL USING (true) WITH CHECK (true);
